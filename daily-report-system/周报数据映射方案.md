### 周报显示内容
10 行：序号、计划事项及完成情况、责任人

```javascript
      taskName: '天花吊顶龙骨安装',
      progress: '80%',
      owner: '张师傅',
      description: '员工餐厅区天花吊顶龙骨安装',
      headcount: 2
    },
    status: 'confirmed'
  },
  // ... 更多 progress 类型事件
]
```

### 数据映射

| 周报字段 | 现有字段 | 状态 | 提取方式 |
|----------|---------|:----:|----------|
| 区域分组 header | `areaId` + `AREAS[].name` | ✅ | 按 areaId 分组生成 header |
| 计划事项及完成情况 | `payload.taskName` + `payload.progress` | ✅ | 拼接 `taskName + "完成" + progress` |
| 责任人 | `payload.owner` | ⚠️ | 部分 mock 事件缺 owner |

### 聚合伪代码
```javascript
function getPage04Data(projectId, weekStart, weekEnd) {
  const weekEvents = EVENTS.filter(e =>
    e.projectId === projectId &&
    e.type === 'progress' &&
    e.date >= weekStart && e.date <= weekEnd &&
    e.status === 'confirmed'
  )

  // 按 areaId 分组
  const grouped = groupBy(weekEvents, 'areaId')

  const rows = []
  Object.entries(grouped).forEach(([areaId, events]) => {
    const area = getProjectAreas(projectId).find(a => a.id === areaId)
    // 区域 header 行
    rows.push({ type: 'header', text: `${area.name}：` })
    // 具体事项行
    events.forEach((e, i) => {
      rows.push({
        type: 'detail',
        seq: i + 1,
        text: `${e.payload.taskName}完成${e.payload.progress}`,
        owner: e.payload.owner || '—'
      })
    })
  })
  return rows
}
```

| 周报字段 | 现有字段 | 状态 | 提取方式 |

### 新增建议

**扩展 ISSUES type='ecc' 的语义：** 将 ECC 作为一个独立跟踪实体，而非只是 ISSUES 的子类型。建议新增：

```javascript
ECC_ITEMS = [
  {
    id: 'ECC001',
    projectId: 'baicaoyuan',
    title: '问题描述',
    areaId: 'A2',
    discoveredDate: '2026-04-15',
    status: 'closed',  // 'open' | 'in_progress' | 'closing' | 'closed'
    closingProgress: '流程关闭中',  // 仅 status='closing' 时
    closedDate: '2026-05-10',
    photos: ['P003']
  }
]
```

**聚合逻辑：**
```javascript
function getECCStats(projectId) {
  const eccs = ECC_ITEMS.filter(e => e.projectId === projectId)
  const total = eccs.length
  const closed = eccs.filter(e => e.status === 'closed').length
  const closing = eccs.filter(e => e.status === 'closing').length
  const open = eccs.filter(e => e.status === 'open' || e.status === 'in_progress').length
  const rate = total > 0 ? ((closed / total) * 100).toFixed(2) + '%' : '—'
  return { total, closed, closing, open, rate }
}
```

## 总结：新增实体与字段清单

| 06 | 扩展字段 + 标准化 | 新增 `nextWeekCount` 到周计划；标准化工种名 | **高** |
